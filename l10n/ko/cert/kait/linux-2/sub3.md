
# 3과목: 리눅스 활용

* **3과목: 리눅스 활용**
  * 1항목: X 윈도우
    * 개념 및 사용법

  * 2항목: 인터넷 활용

## X 윈도우 개념

* 리눅스 커널을 기반으로 GUI를 제공
* 플랫폼과는 독립적으로 작동하고, KDE, GNOME 등이 있음
  * 즉, 배포판을 가리지 않고 KDE, GNOME 등을 사용 가능
